# personal-enterprise-lab
## Overview

This repository documents the design, implementation, and maintenance of my personal enterprise home lab.

The goal of this project is to gain practical experience with enterprise IT infrastructure, systems administration, networking, virtualization, cybersecurity, and automation using real hardware.

This lab will serve as both a learning platform and a portfolio project.

---

## Objectives

- Build a virtualized infrastructure using Proxmox
- Configure enterprise networking
- Deploy Windows Server and Active Directory
- Learn Linux server administration
- Host Docker containers
- Build centralized storage
- Implement backups and snapshots
- Deploy monitoring and logging
- Practice cybersecurity techniques
- Document every major decision
